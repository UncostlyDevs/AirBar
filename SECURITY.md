# Security Policy

## Supported Versions

Security fixes are handled for the latest public AirBar source release.

## Reporting A Vulnerability

Use GitHub issues or the repository security advisory flow once configured.

Include:

- A short description of the issue.
- Steps to reproduce it.
- The AirBar version or commit you tested.
- Any relevant Windows version details.

Please do not publish exploit details before there has been a reasonable chance to review and fix the issue.

## Security Notes

- AirBar does not require administrator privileges.
- AirBar stores local settings and history under `%AppData%\AirBar`.
- AirBar does not collect telemetry.
- Prebuilt EXE releases should be distributed with a SHA256 checksum.
